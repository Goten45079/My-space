------------------------------------- VPC ----------------------------------------------

VPC (virtual private cloud) -- defining the range of IP addresses -- count 256*256 in number

subnets -- dividing the range of IP addressed and each range is called as subnet

route table -- defines the where network traffic is directed

------------------------------------------- security group ---------------------------------
SG and NACL(network access control list) -- these two are the security settings
SG (security group) is applied at the instance level
NACL (network access control list) is applied at the subnet level

if some security is defined at subnet level(NACL) by default it will be applied to all the instances in side the subnet

---------------------------------- route53 ------------------------

route 53 will provide DNS(domain naming system) as a service 

this service is for domain regesteration and hosted zones and health checks
